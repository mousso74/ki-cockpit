Problem beschreiben

Beschreibe dein Problem so detailliert wie möglich. Die KIs werden zuerst klärende Fragen stellen, dann eigenständig Lösungen erarbeiten, und sich anschließend gegenseitig kritisch bewerten.

Ablauf: Erweiterte Analyse

① Fragen-Prompts generieren → an alle 3 KIs schicken
② KI-Fragen zusammenführen → beantworten
③ Lösungs-Prompts → 3 unabhängige KI-Lösungen
④ Cross-Review → jede KI bewertet die anderen beiden
⑤ 3 überarbeitete Lösungen → finale Synthese

Phase 3 – Cross-Review Prompts

→ das Gute übernehmen  → das Schlechte/Falsche benennen  → eine verbesserte Gesamtantwort erstellen
